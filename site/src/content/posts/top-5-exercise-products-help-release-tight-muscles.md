Five familiar tools can be used in less conventional ways to support mobility:

1. **Kettlebell:** use the handle as a stable point for a hip-flexor release.
2. **Foam roller:** actively target tight areas rather than simply rolling over them.
3. **Barbell:** use the bar as an upper-body mobility aid for tight upper traps and shoulders.
4. **Resistance band:** add variable tension to movements and help position the shoulder girdle.
5. **Dumbbells:** use a neutral load to activate scapular stabilisers and support posture.

Apply only the pressure you can control and seek professional guidance for persistent pain.
